This matrix also yields an interesting link to a widely used value enhancement measure—economic
value added (EVA). One of the biggest sales pitches for EVA, which is computed as the product of
the return spread (ROC minus cost of capital) and capital invested, is its high correlation with market
value added (MVA) which is defined as the difference between market value and book value of
capital. This is not surprising, since MVA is a variant on the value-to-book ratio and EVA is a
variant on the return spread.
