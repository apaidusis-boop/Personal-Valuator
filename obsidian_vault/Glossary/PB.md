# 📖 P/B (P/VP) — Price / Book Value

> Categoria: **valuation**. Cross-links: [[CONSTITUTION]] · [[Glossary/_Index]]

## Fórmula

`Preço da acção ÷ Patrimônio líquido por acção (BVPS)`

## Leitura

Quanto o mercado paga por cada R$1 de patrimônio líquido contábil. P/B = 1 → preço = book value; P/B < 1 → desconto sobre book.

## Thresholds

- **br_equity**: Componente do Graham (preço ≤ √(22.5 × EPS × BVPS))
- **br_banks**: ≤ 1.5 (margem de segurança sobre o equity)
- **us**: ≤ 3 (Buffett quality)

## Bom vs Mau

**P/B < 1** sugere desconto, mas pode reflectir activos não-líquidos, goodwill inflado, ou ROE baixo. **P/B alto** justificável só se ROE for elevado (ex: ITUB4 P/B 2.4 com ROE 21%).

## Contraméricas (quando o sinal falha)

❌ Empresas asset-light (software, consultoria, marcas) têm book value subestimado — P/B perde significado. Comparar com peers do mesmo modelo.
❌ Bancos com goodwill grande de M&A (ex: pós-aquisição) inflam book artificialmente — usar P/Tangible-B em vez de P/B.
