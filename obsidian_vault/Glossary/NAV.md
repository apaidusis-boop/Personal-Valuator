# 📖 NAV — Net Asset Value (Patrimônio Líquido)

> Categoria: **valuation**. Cross-links: [[CONSTITUTION]] · [[Glossary/_Index]]

## Fórmula

`Activos − Passivos. Para FIIs: valor dos imóveis + caixa − dívida.`

## Leitura

**Valor patrimonial real** dos activos. Para FIIs, comparar com cota = preço/NAV (P/VP) — cota < NAV = desconto sobre tijolo. Para holdings, soma de partes (sum-of-parts) = NAV implícito.

## Thresholds

- **fii_desconto**: Cota / NAV < 0.95 = desconto significativo
- **fii_premio**: Cota / NAV > 1.10 = pago premio (justifica só com qualidade)
- **holding_disc**: ITSA4 historicamente 10-25% desconto vs NAV (Itaú+participações)

## Bom vs Mau

**Desconto persistente** = oportunidade OU bandeira vermelha (Mr Market vê algo). **Premio** = margem de erro pequena, exigir qualidade extrema.

## Contraméricas (quando o sinal falha)

❌ NAV é estimativa — laudos de imóveis lagam mercado.
❌ Holdings: NAV ignora desconto burocrático/governance que persiste.
❌ NAV ajustado por goodwill é mais conservador (deflate intangible).
