## рҹ“¬ Canais de comunicaГ§ГЈo

### Inbound (agents aprendem do mundo)
- **Fetchers** вҖ” brapi, yfinance, FRED вҶ’ scheduled cron
- **Scrapers** вҖ” Fool/XP/WSJ via Sofia Clippings + Playwright persistent
- **Monitors** вҖ” CVM/SEC via Ulisses Navegador
- **YouTube** вҖ” on-demand `ii ingest <url>`

### Internal (agents falam entre si via DB)
- `analyst_reports` (extracГ§ГЈo Wilson) вҶ’ `analyst_insights`
- `analyst_insights` вҶ’ Clara Fit вҶ’ `watchlist_actions`
- `watchlist_actions` вҶ’ Aurora Matina resume вҶ’ briefing
- `analyst_insights` вҶ’ AristГіteles Backtest вҶ’ `predictions` вҶ’ credibility
- `agents/*.json` state вҶ’ Regina Ordem вҶ’ dashboard
- `predictions` вҶ’ Clara Fit weight adjustment

### Outbound (agents falam ao founder)
- **Telegram push** вҖ” Aurora (daily), Wilson (alerts), Valentina (risk), Regina (cohort unhealthy)
- **Obsidian vault** вҖ” persona status pages, dashboards, briefings, wiki updates
- **Telegram pull (two-way)** вҖ” ZГ© Mensageiro recebe /status /run /approve

### Return path (founder вҶ’ sistema)
- Telegram `/approve <id>` вҶ’ ZГ© Mensageiro вҶ’ `UPDATE watchlist_actions`
- Manual edits em wiki/holdings/*.md вҶ’ prГіxima run de Teresa Tese preserva entre markers

## рҹҡҰ Criticidade dos fluxos

| Fluxo | Se falhar | Severity |
|---|---|---|
| brapi/yfinance fetch | Dados stale вҶ’ decisГөes com info antiga | рҹ”ҙ Alta |
| Sofia Clippings | Weekly reports atrasados | рҹҹЎ MГ©dia |
| Wilson auto-extract | Insights pending acumulam | рҹҹЎ MГ©dia |
| Aurora briefing | Founder acorda sem sinopse | рҹҹў Baixa (pode correr manual) |
| Valentina drift | Holding drift nГЈo detectado | рҹ”ҙ Alta |
| Regina Ordem | Agents failing sem auto-disable | рҹҹЎ MГ©dia |
| ZГ© Mensageiro | Founder perde canal remote | рҹҹЎ MГ©dia |
