# 🔌 Semi Cycle — framework

## Fases canónicas

```
Shortage → Capacity Chegando → Oversupply → Consolidação
   │            │                  │             │
ASP ↑        ASP flat           ASP ↓↓         ASP bottom
Boosters     Neutralized       Stressed       Recovery
```

## Fase 1: Shortage / Boom

**Sinais**:
- Lead times 6-12m+ across categories
- ASP rising across memory + logic
- OEM orders outstanding grows
- Fabs full utilization (95%+)
- Capex announcements massive (TSMC +$30B, Intel +$50B)

**Duration**: 12-24 meses.
**Winners**: Everyone. Fabless + foundry + equipment.
**Caveat**: Multiple expansion → **watch for peak P/E trap**.

**Historical**:
- 2000 dot-com (pré-bust)
- 2017-2018 (AI-first wave, crypto GPU demand)
- 2020-2022 (pandemic WFH + crypto + supply chain constraints)
- 2023-2025 (AI Gen-AI infrastructure)

## Fase 2: Capacity chegando

**Sinais**:
- New fabs completing (2y post-announcement)
- Customer inventory building excess
- Lead times normalising (6m → 3m)
- Sell-through softening
- ASP plateau

**Duration**: 6-12 meses.
**Winners**: None outperform. Watch for early-cycle distortion.
**Action**: **Trim winners**, especially commodity-sensitive (memory, consumer logic).

## Fase 3: Oversupply / correção

**Sinais**:
- Customer inventory > 120 days (DOH)
- Revenue guidance cuts (ASML bookings softness warning)
- Layoffs announced
- Memory prices crash 30%+
- Capex cuts (intel paused Ohio mid-2023)

**Duration**: 3-8 quarters (varies by segment).
**Winners**: **None**. Sector stress testing.
**Losers**: Memory primeiro (MU, Hynix), then commodity logic.

**Historical**:
- 2001-2002 (dot-com bust)
- 2008-2009 (GFC demand collapse)
- 2018-2019 (post-crypto, GDDR5 glut, Huawei ban)
- 2022-2023 (post-COVID inventory destocking)

## Fase 4: Consolidação / recovery

**Sinais**:
- Inventories bottom (DOH < 70 days)
- Pricing stabilise, no more cuts
- Fabs utilization recovering (< 85% → 90%+)
- Capex resumed with discipline
- M&A activity (weak consolidate into stronger)

**Duration**: 6-12 meses.
**Winners**: Best balance sheets buying into recovery (TSM steady, NVDA cash pile).
**Action**: **Best entry** — position for next shortage.

**Historical**:
- 2003-2004 (post-dot-com reset)
- 2010 (post-GFC mobile-Android wave)
- 2020 (post-COVID panic → 5G/WFH thesis)
- 2024 (AI recovery begin)

## Segmentos internos — desincronizados

### Memory (DRAM/NAND) — mais volátil
- Cycle mais rápido, 18-24m.
- Spot prices daily available → transparent timing.
- 2022-2023 DRAM -50% troughs.
- 2024-2025 recovery HBM (AI driver).

### Logic (CPU/GPU)
- Cycle 3-4 anos.
- Specialised by vertical (datacenter GPU, mobile SoC, PC CPU).
- NVDA datacenter em super-cycle 2023-2026.

### Analog (TXN, ADI)
- Cycle menos pronunciado, 3-5y lento.
- Diversified content increasing (cars, industrial).
- DRIP-friendly (TXN aristocrat).

### Equipment (ASML, AMAT, LRCX)
- Cycle lags chip cycle 2-4 quarters (order book dynamic).
- ASML 2025 backlog massive → less cyclical appearance.

### EDA (SNPS, CDNS)
- Quasi-subscription → very low cyclicality.
- R&D spend of fabless clients = their revenue.

## Capital intensity reality

Every fab cycle = billions $ locked in:
- 3nm fab: $20-30B capex, 2-3y to first wafer.
- Depreciation 5-8y → margin pressure years 1-3.
- Once ramped, margin contribution 30-40%+ incremental.

**Implication**: **FCF lumpy**, normalised valuation essential.

## Red flags — cycle turning bearish

- DRAM DOH > 130 days → correction coming 6m
- SEMI billings B/B < 0.95 sustained → order book collapse
- TSM utilisation guidance cut → downstream demand weakening
- Hyperscaler capex guidance cut > 10% YoY

## Red flags — cycle turning bullish

- Inventories bottomed + 2 quarters declining
- OEMs restocking (phones, PCs)
- Memory pricing +10% MoM sustained
- Fab utilisation > 90% across top-3 foundries

## Playbook application

- **Fase 1 (Shortage)**: **TRIM winners** near peak (avoid "this time different").
- **Fase 2 (Capacity chegando)**: **reduce exposure** memory + commodity logic.
- **Fase 3 (Oversupply)**: **accumulate quality** balance sheets (TSM, TXN, AVGO).
- **Fase 4 (Recovery)**: **ADD positions**, position for shortage.

Semi is **not** a DRIP-pure sector except specific aristocrats (TXN, AVGO). Tactical allocation max 10% teu portfolio.
