# Graham Deep Value

## A filosofia

Graham **não procura empresas boas baratas** — procura **empresas mediocres/ruins suficientemente baratas**. O alfa vem da margem de segurança estatística, não da qualidade do negócio.

## Os 3 pilares

### 1. Margem de segurança
Comprar só com **desconto ≥ 30-50%** vs intrinsic value (seja Graham Number, NAV, earnings power).

### 2. Diversificação estatística
Cesta de **20-30 deep value** — alguns vão à falência, mas em média supera.

### 3. Estudo quantitativo, não qualitativo
Graham era **cético** de forecasts, guidance, "moats". Confia só em **números reportados**.

## Critérios clássicos (Intelligent Investor cap. 14)

| Critério | Threshold |
|---|---|
| Revenue | ≥ $100M (hoje: $500M+) |
| Current ratio | ≥ 2 |
| Lucro positivo | 10 anos consecutivos |
| Dividendos | 20 anos consecutivos |
| EPS growth | ≥ 33% em 10 anos |
| P/E | ≤ 15× média 3y |
| P/B | ≤ 1.5 |
| **P/E × P/B** | **≤ 22.5** ([[Graham_Number]]) |

## Net-net (mais extremo)

Comprar por **< 2/3 do NCAV** (Net Current Asset Value):

$$
\text{NCAV} = \text{Activos correntes} - \text{Passivos totais}
$$

Praticamente só existe em **micro-caps distressed** ou crashes (2008, 2020).

## Relação com [[Buffett_quality]]

Buffett começou Graham, evoluiu: *"É melhor comprar uma empresa maravilhosa a preço justo do que uma justa a preço maravilhoso"*. Mas a **margem de segurança** (núcleo Graham) nunca deixou.

## Quando DEEP VALUE funciona historicamente

- **Pós-crashes**: 1974, 2002, 2009, 2020 — reversal violento
- **Setores odiados**: tabaco 2000s, carvão 2016, BR 2016 (Dilma)
- **Regime value > growth**: 1960-80s, 2000-2007, 2022-hoje(?)

## Quando DEFAULT (value trap)

- **Secular decline**: newspapers, TV linear, retail físico (MACY crashed do "cheap")
- **Accounting fraud**: os números mentiram (Enron, Valeant, WIRE)
- **Leverage hidden**: Sears parecia cheap, morreu a debt

## No nosso sistema

- Screen BR: Graham Number + DY 6% + ROE 15% + NetDebt<3 + streak 5y
- Veto Altman < 1.81 → evita value traps distressed
- Veto Piotroski ≤ 3 → evita deterioração cumulativa
